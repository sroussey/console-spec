The console object

This section defines the console object.

void log (Object object)
Writes a message to the console
void debug (Object object)
Writes a message to the console, along with the line number for the line where it was called.
void info (Object object)
Writes a message to the console, with a flag of some kind to indicate that it is an informational message, along with the line number for the line where it was called.
void warn (Object object)
Writes a message to the console, with a flag of some kind to indicate that it is a warning message, along with the line number for the line where it was called.
void error (Object object)
Writes a message to the console, with a flag of some kind to indicate that it is an error message, along with the line number for the line where it was called.
void assert (boolean expression, optional Object object)
  1. Tests whether the expression argument evaluates to true.
  2. If the expression argument does not evaluate to true:
    1. Throws an exception.
    2. Optionally, writes a message to the console.
