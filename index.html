The Console Interface represents a means for programatically writing output to consoles such as the console environments included as part of Web-developer toolsuites in Web browsers, and shell environments available for various scripting languages.

Additional editorial contributions to this document are welcome. For details on how to contribute, see the instructions in the accompanying README file.

The console object

This section defines the console object.

void log (in any... args)
Outputs log content args to the console
void debug (in any... args)
Outputs log content args to the console, along with the line number for the line where it was called.
void info (in any... args)
Outputs log content args to the console, with a flag of some kind to indicate that it is an informational message, along with the line number for the line where it was called.
void warn (in any... args)
Outputs log content args to the console, with a flag of some kind to indicate that it is a warning message, along with the line number for the line where it was called.
void error (in any... args)
Outputs log content args to the console, with a flag of some kind to indicate that it is an error message, along with the line number for the line where it was called.
void assert (boolean expression, in any... args)
Tests whether the boolean expression argument evaluates to true; and, if expression does not evaluate to true, throws an exception and, optionally, outputs log content args to the console.
